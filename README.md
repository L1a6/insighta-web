# Insighta Web Portal

A lightweight web portal for Insighta Labs+.

## Setup

```bash
npm install
```

Create a `.env` file:

```bash
VITE_API_BASE_URL=http://localhost:3000
```

## Run

```bash
npm run dev
```

## Pages

- Login (GitHub OAuth)
- Dashboard (metrics)
- Profiles list (filters + pagination)
- Profile detail view
- Search
- Account
